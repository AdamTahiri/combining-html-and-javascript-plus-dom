Combining HTML and JavaScript + DOM Lab

Complete each of the sections below

1. String Mirror

Make a form with id='mirror-form'

The user should be able to enter a string into a text input with id='mirror-input'

They should then be able to click a "submit" button that will display the string the user entered in an element with id='mirror-output'

2. String Uppercaser

Make a form with id='uppercaser-form'

The user should be able to enter a string into a text input with id='uppercaser-input'

They should then be able to click a "submit" button that will display the string the user entered in all uppercase in an element with id='uppercaser-output'

3. Palindrome Detector

Make a form with id='palindrome-form'

The user should be able to enter a string into a text input with id='palindrome-input'

They should then be able to click a "submit" button that will display a string in the form "It is ${true/false} that ${entered string} is a palindrome" with id='palindrome-output'

4. Even Checker

Make a form with id='even-checker-form'

The user should be able to enter a number into an input with id='even-checker-input'

They should then be able to click a "submit" button that will display a string in the form "It is ${true/false} that ${entered number} is even" with id='even-checker-output'

5. Number doubler

Make a form with id='doubler-form'

The user should be able to enter a number into an input with id='doubler-input'

They should then be able to click a "submit" button that will display a string in the form "The ${entered number} doubled is ${doubledVal}" with id='even-checker-output'

6. Average of Three Numbers

Make a form with id='average-form'

The user should be able to enter 3 numbers into text inputs with ids 'average-input-1', 'average-input-2', and 'average-input-3

They should then be able to click a "submit" button that will display a string in the form "The average of ${numberOne}, ${numberTwo}, and ${numberThree} is ${average}" with id='average-output'

Bonus: Vowel Remover

Make a form with id='vowel-remover-form'

The user should be able to enter a string into a text input with id='vowel-remover-input'

The user should be able to select or deselect a checkbox with id='y-is-vowel-checkbox'

They should then be able to click a "submit" button that will display the original string with all vowels removed with id='average-output'. If the checkbox is checked, count y as a vowel. Otherwise, count y as a consonant.
